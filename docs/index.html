DebridDownloader

Blazing-fast desktop client for Real-Debrid

Manage torrents and downloads with a native app built on Rust + React

Download on GitHub Other platforms ↓

Features

Torrent Management

Add magnets or .torrent files, select specific files, and monitor progress in real time.

Tracker Search

Add your own tracker sources in settings and search across all of them in parallel.

Download Engine

Multi-threaded downloads with real-time speed, ETA, and progress tracking.

Keyboard First

⌘K search, ⌘R refresh, arrow navigation, Enter to download. Full keyboard control.

Secure Storage

API tokens stored in OS keychain. macOS builds signed and notarized by Apple.

Themes

Dark and light mode with 6 accent colors. Runs quietly in your system tray.
